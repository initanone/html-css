MACACOS SÃO AMIGOS DA GENTE

Macaco é um termo utilizado para referir-se a espécies de primatas. Os macacos são animais importantes para o meio ambiente, estando muitas espécies ameaçadas.

"Como mencionado, os macacos são animais pertencentes à ordem Primates, assim como os seres humanos. Dentre as principais características dos primatas, podemos destacar o cérebro bem desenvolvido, os olhos voltados para a frente, e o chamado polegar opositor.
O polegar opositor nos macacos é importante para que consigam agarrar objetos, funcionando como verdadeiras pinças."
